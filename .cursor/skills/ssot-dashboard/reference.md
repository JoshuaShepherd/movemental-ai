## Six-Layer Type Safety Chain

Layer 1: Database (Drizzle) → Layer 2: Zod schemas → Layer 3: Services → Layer 4: API routes → Layer 5: Hooks → Layer 6: UI. Types flow downstream only.

## Design Principles (from UI/UX proposal)

Progressive disclosure, visual hierarchy, reading-optimized layout, trust through transparency, mobile-first. Card-based layouts, step indicators, clean typography, token-based styling.
