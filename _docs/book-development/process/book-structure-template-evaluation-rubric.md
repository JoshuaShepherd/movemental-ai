# Book Structure Template Evaluation Rubric

**Date**: January 2026  
**Purpose**: Evaluate which book structure templates best fit your book vision  
**Prerequisites**: Complete the Book Vision Development Wizard first

---

## How to Use This Rubric

This rubric evaluates each of the 12 book structure templates against your completed vision. For each template, score how well it aligns with your vision across multiple dimensions.

**Process**:
1. Complete the Book Vision Development Wizard
2. Review your vision elements
3. Score each template (1-5 scale) on each dimension
4. Calculate total scores
5. Identify top 2-3 templates for further consideration

**Scoring Scale**:
- **5 = Excellent Fit**: Template aligns perfectly with this vision element
- **4 = Good Fit**: Template aligns well with this vision element
- **3 = Moderate Fit**: Template somewhat aligns, some adjustments needed
- **2 = Poor Fit**: Template doesn't align well with this vision element
- **1 = Very Poor Fit**: Template conflicts with this vision element

---

## Vision Elements Reference

Before scoring, review your vision wizard answers for:

- **Core Message & Purpose**: What you're saying and why
- **Target Audience**: Who you're writing for
- **Content Type**: Framework, paradigm shift, practical guide, etc.
- **Desired Outcomes**: What transformation you want
- **Voice & Style**: How you want to communicate
- **Structure Needs**: Required structural elements
- **Resources Available**: What you have to work with
- **Constraints**: Limitations you face
- **Scope & Scale**: How big and broad
- **Unique Characteristics**: What makes it distinctive

## Scoring Calculation Reference

**Weighted Score Formula**:
```
Weighted Score = (
  (Content Type Alignment × 0.20) +
  (Core Message Support × 0.20) +
  (Target Audience Fit × 0.15) +
  (Desired Outcomes Alignment × 0.15) +
  (Resource Utilization × 0.10) +
  (Constraint Accommodation × 0.10) +
  (Voice & Style Compatibility × 0.10)
)
```

**Interpretation**:
- **4.5-5.0**: Excellent fit, use this template
- **3.5-4.4**: Good fit, consider this template
- **2.5-3.4**: Moderate fit, may need significant adaptation
- **1.5-2.4**: Poor fit, not recommended
- **1.0-1.4**: Very poor fit, avoid this template
