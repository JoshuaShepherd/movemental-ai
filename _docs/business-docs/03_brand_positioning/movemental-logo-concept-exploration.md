## Usage Guidelines

### How to Use These Prompts

1. **Generate All 25**: Create all 25 concepts to see the full range of visual directions
2. **Compare Concepts**: Look for which concepts reveal the strongest design philosophy
3. **Identify Patterns**: Notice which visual metaphors resonate most
4. **Select Direction**: Choose 3-5 concepts that best represent Movemental's identity
5. **Refine Selected**: Use selected concepts as foundation for final logo development

### What to Look For

**Strong Concepts Will Show**:
- Clear visual metaphor that communicates brand values
- Typography that matches brand personality
- Color relationships that feel cohesive
- Composition that suggests the full design language
- Scalability and versatility
- Distinctiveness from competitors

**Weak Concepts Will Show**:
- Overly literal interpretations
- Generic or clich√© visual metaphors
- Typography that doesn't match brand
- Color relationships that feel disconnected
- Composition that doesn't suggest broader design language
- Lack of scalability or versatility

### Next Steps After Generation

1. **Review All 25**: Assess which concepts reveal strongest design directions
2. **Select Top 5**: Choose concepts that best represent Movemental
3. **Develop Full System**: Use selected concepts to develop complete design system
4. **Refine and Iterate**: Refine selected concepts based on results
5. **Build Out**: Develop full visual identity from selected concepts
