# Network Page (`/network`)

## Purpose
Showcase curated Movemental leaders and organizations, reinforce scarcity, and provide structured exploration paths across archetypes, geographies, and initiatives.

## Audience
- Referred leaders verifying peer presence.
- Partners seeking credible examples (Forge, Wesleyan, Tampa Underground).
- Media/investors wanting proof of traction.

## Outcomes
1. Visitors explore network members via filters (archetype, region, initiative).
2. Users discover case studies (Dave, Soul of Movement, Forge America).
3. Page prompts referrals via Movemental Gate CTA.

## Component Map
1. **Intro Banner**
   - Copy: “Movemental is a curated scenius of movement leaders.”
   - Stats: # of active leaders, orgs, research hubs.
2. **Filter Toolbar**
   - Dropdowns: Leader archetype (Platform Architect, Network Amplifier, Emerging Storyteller, Research Partner), Geography, Initiative Type.
   - Search bar for names/topics.
3. **Leader Grid**
   - Cards with portrait, name, domain/initiative, 1-line impact metric, CTA (`Visit Platform`, `View Case Study`).
   - Example categories: Alan Hirsch, Dave Ferguson, Rob + Brian, Deb Hirsch/Mandy/Bree, Bree Mills, Geoff Maddock, Daniel Yang.
4. **Organization Spotlights**
   - Horizontal cards for Forge America, Forge Canada, Wesleyan Multipliers Journey, Tampa Underground.
   - Each card lists deployment scope, shared resources, CTA to request briefing.
5. **Initiative Carousel**
   - Rotating stories (Soul of Movement microsite, Send Institute research hub, Content Studio sprints).
6. **Pipeline Snapshot**
   - Visual showing number of leaders/orgs in Engage / Prototype / Nurture status.
   - Links to Gate page.
7. **Referral CTA**
   - “Know a Movemental leader?” summary of rubric + button linking to secure referral form.
8. **Footer CTA**
   - `Return to Story`, `See Media Lab`, `Contact Movemental`.

## Data Needs
- CMS entries for each leader/org (name, bio, archetype, readiness status, URLs, imagery).
- Pipeline metrics from internal CRM.
- Case study links (Dave launch, Soul of Movement, Forge America, Send Institute, Geoff story sprint).

## Tone & Interaction
- Confident, curated, no generic “directory” feel.
- Hover interactions reveal added metrics (e.g., “90% revenue retained”, “28x amplification achieved”).
- Apply design rules for contrast/accessibility.
