# Author Profile Presentation Standards

> **Visual, Interpretational, Concise Author Intelligence** - Presentation standards and agent executable prompt for creating author profiles that prioritize insight over information, visual clarity over prose, and strategic interpretation over raw data.

**Version**: 1.0.0  
**Last Updated**: January 17, 2026  
**Purpose**: Defines presentation format for author profiles that are scannable, actionable, and visually engaging while maintaining programmatic access for tool integration.

---

## Core Philosophy

**Author profiles should be:**

1. **Interpretational First** - Lead with strategic insights, not raw data
2. **Visual & Scannable** - Use visual elements (tables, scorecards, indicators) for quick comprehension
3. **Concise & Actionable** - Every element serves a strategic purpose
4. **Programmatically Accessible** - Structured data remains extractable for tool integration

**Word Count Philosophy**: Instead of fixed word counts, aim for optimal information density:
- **40% Visual/Structured** (tables, scorecards, matrices)
- **60% Interpretive Prose** (strategic insights, implications)
- **Total Target**: 1,000-1,300 words, but density matters more than count

---

## Profile Structure Template

Every author profile (Author, Audience, Content Audit, Calling) follows this structure:

### 1. **At-A-Glance Scorecard** (Visual Dashboard)
- Quick visual summary of key metrics
- Status indicators (‚úÖ ‚ö†Ô∏è ‚ùå)
- Score/percentage where applicable
- Positioned at the very top for immediate comprehension

### 2. **Strategic Insight** (Interpretative Positioning)
- 2-3 paragraphs of strategic interpretation
- What makes this author unique/distinctive?
- Why this matters strategically
- What this means for content strategy

### 3. **Visual Summary** (Tables, Matrices, Indicators)
- Key data presented in scannable visual format
- Use tables, progress bars, comparison matrices
- Reduce prose, increase visual information density

### 4. **Deep Dive** (Supporting Evidence)
- Detailed sections with supporting data
- Maintains structured format but provides depth
- Supports the strategic insights above

### 5. **Actionable Implications** (Quick Reference)
- Clear, actionable takeaways
- Visual boxes or tables for quick reference
- Directly connects to content strategy
