# Templates

- **TEMPLATE_CONTRACT.md** — Activation, tokens, and scoping contract (v1) for both the in-app Movement Leader template and the static library templates.
- **Template CSS (app):** `app/styles/templates/` — Movement Leader in `movement-leader.css`; library scoped exemplars in `library/<id>.scoped.css`.
- **Static library:** `public/templates/library/<id>/` — Synced from external repo; see `_docs/TEMPLATE_LIBRARY_SYNC.md`.
- **Status and hardening:** `_docs/system-status/TEMPLATE_SYSTEM_STATUS.md`, `_docs/system-status/TEMPLATE_HARDENING_PLAN.md`.
