## Overview

This directory contains the complete system for generating authoritative source documents for Alan's core concepts. These documents serve as the foundation for all derivative content (articles, FAQs, infographics, courses, etc.).

### The Content Chain Vision

```
SOURCE DOCUMENTS (Authoritative Foundation)
    ↓
EXTENDED CONTENT (Derivatives)
    ↓
PRACTICAL APPLICATIONS (Implementation)
    ↓
VISUAL & MULTIMEDIA (Communication)
```

## Document Types

### 1. Master Document
- **Purpose**: Comprehensive authoritative source
- **Length**: 8,000-12,000 words
- **Contents**: Definition, sources, subtopics, scripture index, relationships, history, objections

### 2. Executive Summary
- **Purpose**: Quick reference
- **Length**: 500-800 words
- **Contents**: Core definition, key points, essential quotes, primary sources, related concepts

### 3. FAQ Document
- **Purpose**: Address common questions
- **Length**: 3,000-5,000 words
- **Contents**: 15-25 common questions with comprehensive answers

### 4. Visualization Brief
- **Purpose**: Guide for visual creators
- **Length**: 500-1,000 words
- **Contents**: Visual metaphors, diagram concepts, infographic ideas, layout suggestions

### 5. Implementation Guide
- **Purpose**: Step-by-step application
- **Length**: 2,500-4,000 words
- **Contents**: Assessment, step-by-step implementation, pitfalls, success indicators, case studies

## Quality Standards

### Voice Alignment
- **Requirement**: Coherence score ≥ 0.75
- **Check**: Use voice evaluation system (`_docs/voice/`)
- **Key Markers**: Christocentric anchoring (≥0.7), Prophetic intensity (0.5-0.8), Pastoral warmth (≥0.5), Narrative imagery (≥0.5), Theological depth (≥0.4)

### Theological Accuracy
- All theological claims must be biblically grounded
- Scripture references must be accurate and contextual
- Must align with Alan's theological framework

### Completeness
- All source material must be included
- Every book/article reference must be documented
- No concept should have "gaps" in source coverage

### Citation Standards
- All quotes and references must be properly cited
- Format: Book title, chapter, page (when available)
- Every claim must be traceable to source
