# Content Types: Quintessential Guides

**Purpose**: Complete structural, content, and organizational blueprints for creating content from source material.

**Status**: Active Documentation  
**Last Updated**: December 2025

---

## Overview

This directory contains quintessential guides for each content type available in the platform. Each guide provides a complete blueprint for transforming source material (like master documents) into specific content formats. When you apply a guide to actual source content (like metanoia), the guide informs exactly what goes into that particular content—even if it needs amplification or further research.

---

## Available Content Types

### 1. [Article](./article.md)
**Purpose**: Text-based content optimized for reading

**Best For**:
- Quick reference
- Deep dives on specific topics
- Thought leadership
- SEO-optimized content
- Shareable content

**Key Elements**:
- Title, meta description, excerpt
- Introduction, main sections, conclusion
- Supporting elements (quotes, examples, scripture)
- SEO optimization
- Related content links

**Length**: 500-5,000 words (typically 1,500-2,500)

---

### 2. [Course](./course.md)
**Purpose**: Structured learning experiences for transformation

**Best For**:
- Comprehensive learning
- Skill development
- Transformation journeys
- Structured application
- Certification programs

**Key Elements**:
- Course title, description, learning objectives
- Modules and lessons
- Interactive elements (cohort sessions, field experiments)
- Assessments and resources
- Certificate options

**Length**: 2-26 weeks, 15-60 hours

---

### 3. [Video](./video.md)
**Purpose**: Visual storytelling with audio narration

**Best For**:
- Visual storytelling
- Complex concept explanation
- Personal presence
- Emotional connection
- Demonstration and examples

**Key Elements**:
- Video title, description, script
- Visual plan (slides, graphics, b-roll)
- Introduction, main content, conclusion
- Captions and transcripts
- Thumbnail and metadata

**Length**: 5-60 minutes (typically 15-30)

---

### 4. [Podcast](./podcast.md)
**Purpose**: Audio content optimized for listening

**Best For**:
- Commute listening
- Background learning
- Conversational exploration
- Interview formats
- Extended discussions

**Key Elements**:
- Episode title, description, script/outline
- Show notes
- Introduction, main content, conclusion
- Music and sound effects
- Transcript and metadata

**Length**: 20-90 minutes (typically 30-60)

---

### 5. [Book](./book.md)
**Purpose**: Comprehensive written treatments

**Best For**:
- Authoritative treatments
- Comprehensive frameworks
- Reference materials
- Legacy content
- Deep theological work

**Key Elements**:
- Book title, subtitle, description
- Table of contents
- Introduction, parts, chapters, conclusion
- Appendices, bibliography, indexes
- Supporting elements (quotes, examples, scripture)

**Length**: 20,000-120,000 words (100-600 pages)

---

### 6. [Event](./event.md)
**Purpose**: Time-bound interactive experiences

**Best For**:
- Live teaching
- Interactive workshops
- Community building
- Real-time Q&A
- Cohort experiences

**Key Elements**:
- Event title, description, format
- Schedule and sessions
- Interactive elements (discussion, exercises, Q&A)
- Materials (handouts, slides, resources)
- Follow-up resources

**Length**: 1-5 days (2-40 hours)

---

### 7. [Assessment](./assessment.md)
**Purpose**: Diagnostic and evaluation tools

**Best For**:
- Self-evaluation
- Personalized pathways

- Diagnostic tools

**Key Elements**:

- Questions (multiple choice, Likert scale, open-ended)
- Scoring system
- Results report

---

### 8. [Visualization](./visualization.md)
**Purpose**: Visual representations of concepts

**Best For**:
- Quick understanding

**Key Elements**:
- Visualization title, type, brief

- Layout and design

**Formats**: Infographic, diagram, chart, concept map, timeline, flowchart

## How to Use These Guides

### Step 1: Identify Source Material
- Determine which source material you're working with (master document, FAQ, implementation guide, etc.)

### Step 2: Choose Content Type
- Decide which content type best serves your purpose (article, course, video, etc.)

### Step 3: Review Guide
- Read the complete guide for your chosen content type
- Understand all structural elements required
- Note content approach and voice requirements
- Review organizational perspective

### Step 4: Apply to Source Material
- Extract relevant content from source material
- Structure according to guide specifications
- Add content type-specific elements
- Ensure voice alignment
- Verify all required components

### Step 5: Quality Check
- Use the quality checklist in each guide
- Verify voice markers (coherence ≥0.75)
- Check for prohibited patterns (no antithesis)
- Ensure all elements are present
- Test functionality (links, media, etc.)

---

## Common Elements Across All Types

### Voice Alignment
All content types must maintain voice alignment with these markers:
- **Christocentric Anchoring** (≥0.7)
- **Prophetic Intensity** (0.5-0.8)
- **Pastoral Warmth** (≥0.5)
- **Narrative Imagery** (≥0.5)
- **Theological Depth** (≥0.4)

### Prohibited Patterns
All content types must avoid:
- ❌ Antithesis structures ("not X, but Y")
- ❌ Contrastive language ("far from", "rather than", "instead of")
- ✅ Direct affirmative statements
- ✅ Constructive, forward-moving language

### Source Traceability
All content types must:
- Trace back to source material
- Include proper citations
- Link to related content
- Maintain theological accuracy

- Design infographic with key points

- Feed all metanoia source material to knowledge base
- Configure agent personality and voice
- Test interactions
- Deploy and monitor
