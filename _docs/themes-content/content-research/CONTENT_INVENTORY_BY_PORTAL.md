## Introduction

This document provides a comprehensive inventory of all existing content research files in `_docs/content-research/core-content/`, organized by portal. Each file is categorized by:
- **Portal(s)** it relates to
- **Content Type** (Q&A, concept definitions, framework explanations, etc.)
- **Alignment Status** (ALIGNED / NEEDS_REVISION / NEW_CONTENT_NEEDED)
- **Notes** on portal alignment and gaps

**Content Types:**
- Comprehensive Q&A (`comprehensive-qa/`)
- FAQ Answers (`faq-answers/`)
- Concept Definitions (`concept-definitions/`)
- Framework Explanations (`framework-explanations/`)
- Thematic Deep-Dives (`thematic-deep-dives/`)
- Practical Guides (`practical-guides/`)
- Case Studies (`case-studies/`)
- Connection Maps (`connection-maps/`)
- Contextual Background (`contextual-background/`)
- Topic Clusters (`topic-clusters/`)
- Story Index (`story-index/`)
- Voice & Style Reference (`voice-style-reference/`)

**Alignment Status:**
- **ALIGNED**: Content aligns with portal theme and terminology (may need minor updates)
- **NEEDS_REVISION**: Content relates to portal but needs terminology/content updates
- **NEW_CONTENT_NEEDED**: Content doesn't exist but is needed for portal completeness
