## HTML Format Specifications

All HTML files use:
- Semantic HTML5 structure
- Inline CSS styles for Google Docs compatibility
- Calibri/Arial font family (Google Docs default)
- Color-coded headings and sections
- Tables for structured data
- Clear visual hierarchy
- Print-friendly styling (8.5in max width, 1in margins)
