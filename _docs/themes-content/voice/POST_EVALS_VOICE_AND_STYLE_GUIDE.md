## Common Failure Patterns (From Evals)

### Pattern 1: Missing Voice Markers

**Failure Mode**: Content is accurate but lacks voice markers.

**Symptoms**:
- No first-person narrative integration
- No rhetorical questions
- No extended metaphors
- Reads like a reference manual

**Fix**: Add voice markers throughout:
- Start with "If you've been..." or "Let me walk you through..."
- Include 3+ rhetorical questions per 1000 words
- Weave metaphors throughout
- Use direct address ("you", "we")

---

### Pattern 2: Antithesis Structures

**Failure Mode**: Uses "not X but Y" or similar contrast structures.

**Symptoms**:
- "Not X but Y" patterns
- "X rather than Y" patterns
- Binary oppositions

**Fix**: Use positive, additive language:
- "X-focused approach" instead of "not Y but X"
- "Both X and Y, with X as primary" instead of "X rather than Y"
- "X with Y integrated" instead of "not Y but X"

---

### Pattern 3: Content Accuracy Issues

**Failure Mode**: Theologically incorrect or misrepresents frameworks.

**Symptoms**:
- Incorrect framework definitions
- Misrepresented relationships
- Theologically inaccurate statements

**Fix**: 
- Verify all framework definitions
- Check theological accuracy
- Ensure proper relationship representation
- Cross-reference with source material

---

### Pattern 4: Lack of Narrative Flow

**Failure Mode**: Reads like a list or manual, not a conversation.

**Symptoms**:
- Bullet points as primary structure
- Ideas stated but not developed
- No paragraph flow
- No connection between ideas

**Fix**:
- Convert lists to paragraphs
- Develop each idea fully
- Create smooth transitions
- Build paragraphs on each other
