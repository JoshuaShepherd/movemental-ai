## Critical Constraint

### Antithesis Prohibition

**CRITICAL RULE**: No use of antithesis (contrasting opposites) should be incorporated into the voice profile, even if examples are found in source text.

**Prohibited Patterns**:
- "Not X but Y"
- "X rather than Y"
- "Either X or Y"
- Contrastive statements

**Required Alternatives**:
- Positive framing
- Additive language ("both/and")
- Inclusive formulations
- Expansive vision

**See**: `09-ANTITHESIS-PROHIBITION.md` for complete documentation
