Week 3 · Integration

Anchoring the new rhythm

Turn this week's concept into a practice that can survive the next six months.

The integration rhythm

Pick one rhythm that is already part of your week — the morning commute, the team huddle, the Sunday drive — and anchor the new practice to it.

  1. Name the existing rhythm.
  2. Decide what triggers the new practice within it.
  3. Name one person who will ask you about it in 14 days.
  4. Write the single sentence you will repeat each time the trigger fires.
Why this matters. Without an integration step, the content of the week tends to stay in the head. With it, the content begins to shape the feet.
