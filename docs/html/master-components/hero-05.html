Hero 05 · Gradient + stanza

Fragmentation is not a tech problem first.

It is an information architecture and trust architecture problem — then a tooling problem. This hero pairs a direct claim with a slower stanza for boards who read in pairs.

When every channel speaks in a different voice, the movement cannot remember itself. The work is to gather without flattening.

Soft gradient wash behind the section keeps the page warm while staying duotone — no decorative shadows, only tonal stacking.
