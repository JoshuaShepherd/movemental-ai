Single-file template: duplicate prompts in PROMPTS, open in a browser, copy into Cursor or Claude Code.

    Select a prompt

    Editable. Use copy buttons or select all to paste elsewhere.
    How to add or duplicate prompts

    Edit the embedded PROMPTS array in this file (see bottom script). Each entry needs id, title, version, tags, scopeIn, scopeOut, sources (paths or URLs), and body (template literal). Keep bodies plain text for safest paste into chat. 

    {
  id: "example-id",
  title: "Short title",
  version: "2026-05-10",
  tags: ["copy", "movemental"],
  scopeIn: ["What the agent may change"],
  scopeOut: ["Explicit exclusions"],
  sources: ["docs/research/note.md"],
  body: `Multi-line prompt…`
}
