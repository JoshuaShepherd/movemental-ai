### mDNA (Missional DNA) / Apostolic Genius

| Field | Value |
|-------|-------|
| Introduced | 2006 (*The Forgotten Ways*) |
| Components | (1) Jesus is Lord [gates maturity], (2) disciple making, (3) missional-incarnational impulse, (4) apostolic environment (APEST), (5) organic systems, (6) liminality–communitas |
| Visual / model | Six-element interdependent system; "Apostolic Genius" = latent movement potential when mDNA activates |
| Adoption evidence | Seminal missional-church text; course adoptions (Wheaton MA, seminaries); 1,581+ Goodreads ratings on flagship; Chinese underground church & early-church cases cited |
| Source | [The Forgotten Ways](https://www.amazon.com/Forgotten-Ways-Reactivating-Apostolic-Movements/dp/1587433869), [Baker 2nd ed.](https://bakerpublishinggroup.com/books/the-forgotten-ways-2nd-edition/2679716) |

Hirsch argues every church carries dormant movement DNA recoverable through six interdependent practices; Christological maturity caps the whole system.

### APEST / 5Q

| Field | Value |
|-------|-------|
| Introduced | 2012 (*Permanent Revolution*); scaled 2017 (*5Q*) |
| Components | Apostle, Prophet, Evangelist, Shepherd, Teacher (Ephesians 4:11) as distributed callings and "original intelligence" of the Body |
| Visual / model | Fivefold typology; 5Q assessments; team-design diagnostics |
| Adoption evidence | **150,000+** assessments (5Q Central); Exponential pre-conferences; denominational training (e.g. LMC Anabaptist) |
| Source | [5Q](https://www.amazon.com/5Q-Reactivating-Original-Intelligence-Capacity/dp/0998639303), [5qcentral.com](https://5qcentral.com) |

Western churches are structurally Shepherd–Teacher heavy; recovering APE functions is presented as essential for movement health.

### Christology → Missiology → Ecclesiology

| Field | Value |
|-------|-------|
| Introduced | 2008/2009 (*ReJesus*); runs through corpus |
| Components | Christology determines missiology determines ecclesiology (ordering principle) |
| Adoption evidence | Core argument of *ReJesus*; cited in missional ecclesiology discussions |
| Source | *ReJesus* (100 Movements revised 2022) |
