### The Ecclesial Minimum

| Field | Value |
|-------|-------|
| Introduced | Recurring across podcasts (no single dated origin); articulated in *Shifting Culture* #175 and *Theology in the Raw* #1022 |
| Components | (1) Worship, (2) community, (3) mission — the smallest viable definition of a church; everything else is optional scaffolding |
| Visual / model | — |
| Adoption evidence | Repeated verbatim across nearly every podcast appearance; quoted in Koinesúnē features and host show-notes; underpins Tampa Underground's 100+ microchurches |
| Source | [Shifting Culture #175](https://www.shiftingculturepodcast.com/episodes/episode-175-lucaspulley), [Theology in the Raw #1022](https://theologyintheraw.com/podcast/1022-rethinking-church-through-micro-churches-as-missional-communities-lucas-pulley/) |

The ecclesial minimum reframes "church" as worship + community + mission embodied in an extended spiritual family — anything beyond that minimum is discretionary scaffolding to be justified, not assumed.

### Hubology / Servant-Equipper

| Field | Value |
|-------|-------|
| Introduced | 2024–2025 (with Rob Wegner, Lance Ford, and Brian Sanders) |
| Components | Three hub functions (Equip, Coach, Convene); three leader types (Local Elders / Equipping Servants–*diakonoi* / Apostolic Teams); two-entity org model (Mission Agency + Network of Microchurches) |
| Visual / model | Two-entity org model embodied in the Underground Network executive stack |
| Adoption evidence | *The Starfish and the Church* "Rise of the Servant Equippers" series (Apr–Jun 2025); Hubology Substack; co-credited to the Wegner/Ford/Pulley/Sanders collaboration |
| Source | [starfishyou.substack.com/p/hubology-rise-of-the-equipping-servants](https://starfishyou.substack.com/p/hubology-rise-of-the-equipping-servants) |

Hubology argues that a translocal apostolic "hub" exists to equip, coach, and convene a network of microchurches — serving rather than consolidating power — and distinguishes local elders, equipping servants, and apostolic teams as differentiated leadership lanes.

| Adoption evidence | Central metaphor of the decentralized-church cluster; foundational to the Starfish and the Church podcast ecosystem |

### The Granny Shot / Threshold Model (applied)

| Field | Value |
|-------|-------|
| Introduced | 2023 (*Missional Culture & The Granny Shot*) |
| Components | Basketball's underhand "granny shot" (Rick Barry / Wilt Chamberlain) + Mark Granovetter's threshold model of collective behavior; culture beats programs; leaders must lower the behavioral threshold for missional practice |
| Visual / model | — |
| Adoption evidence | Single essay; "culture eats programs for breakfast" is the pull-quote |
| Source | [lucaspulley.medium.com/missional-culture-the-granny-shot](https://lucaspulley.medium.com/missional-culture-the-granny-shot-7302272c5bcb) |

His clearest cross-disciplinary move: importing Granovetter's threshold model to argue that cultures (not vision-casting) shape behavior, so leaders must make missional practice socially normal — "look dumb together."

### First-Order vs. Second-Order Power (multicultural church)

| Field | Value |
|-------|-------|
| Introduced | 2019 (*The Parable of the White Pastor*) |
| Components | Distinguishes giving away representation (first-order) from relinquishing actual narrative, budget, and decision-making power (second-order); multicultural community "must also be minority led" |
| Visual / model | — |
| Adoption evidence | One Missio Alliance essay, widely re-shared; one of the most theologically serious pieces in his corpus |
| Source | [missioalliance.org/the-parable-of-the-white-pastor](https://missioalliance.org/the-parable-of-the-white-pastor/) |

Argues that programmatic multiculturalism without power redistribution is hollow — "you've given away representation but… not much power."
