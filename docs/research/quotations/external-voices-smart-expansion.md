# External Voices — Smart Expansion

A second, expanded tier of verified quotations for Movemental — the kind of sharp, sideways, surprising lines Alan Hirsch himself reaches for as chapter epigraphs. Every quote below has been cross-checked against an authoritative source: a primary text, reputable publisher, archival scan, or well-documented academic citation. Where a line is commonly misattributed, we note it and either corrected or skipped it.

**Grouping:** The ten Movemental themes, to match `external-voices-raw.md` and `quotations.md`.

**Scope rule:** Authors not already covered in the existing library — so the McLuhan / Postman / Ellul / Berry / Wright / Newbigin / Willard / Bonhoeffer / Dillard / Robinson / Merton / Lewis / Chesterton / Naval / Godin / Collins / Kelly (1K fans) / Crouch / Chambers / Elliot cluster is intentionally absent here. A handful of *additional* McLuhan / Postman / Ellul / Brueggemann / Le Guin / Kelly / Hopkins lines are included because the existing entries are shallow relative to the size of these authors' quotable corpora.

## 1. Movement & Multiplication

> "A complex system that works is invariably found to have evolved from a simple system that worked. A complex system designed from scratch never works and cannot be patched up to make it work. You have to start over with a working simple system."
> — **John Gall**, *Systemantics: How Systems Really Work and How They Fail* (Quadrangle/The New York Times Book Co., 1977)
> Verified via: Wikipedia / Wikiquote scans of first-edition text (https://en.wikipedia.org/wiki/John_Gall_(author)) and (https://en.wikiquote.org/wiki/John_Gall)
> Themes: movement-multiplication, scale-simplicity-small

> "If evolution almost always occurs by rapid speciation in small, peripheral isolates, then what should the fossil record look like?"
> — **Stephen Jay Gould & Niles Eldredge**, "Punctuated Equilibria: An Alternative to Phyletic Gradualism" (1972), collected in Schopf, ed., *Models in Paleobiology*
> Verified via: TalkOrigins archival text (http://www.talkorigins.org/faqs/punc-eq.html) and PBS *Evolution* library (https://www.pbs.org/wgbh/evolution/library/03/5/l_035_01.html)

## Standout quotes — the ones a marketer would actually pull

These are the sharpest, scroll-stop-grade lines in the set — the ones best fit for hero sections, pull quotes, and chapter epigraphs:

1. **Ursula K. Le Guin (2014)** — "We live in capitalism. Its power seems inescapable — but then, so did the divine right of kings. Any human power can be resisted and changed by human beings." *The crown jewel. Works for any movement framing.*

2. **John Gall (1977)** — "A complex system that works is invariably found to have evolved from a simple system that worked. A complex system designed from scratch never works and cannot be patched up to make it work." *Gall's Law. Devastating to institutional program-building.*

3. **Walter Brueggemann (1978)** — "Our culture is competent to implement almost anything and to imagine almost nothing." *Pierces the whole era in sixteen words.*

4. **Donella Meadows (2008)** — "Systems can't be controlled, but they can be designed and redesigned." *The mission-network mindset in one line.*

5. **Simone Weil (1942)** — "Attention is the rarest and purest form of generosity." *The attention-economy antidote.*

6. **James P. Carse (1986)** — "A finite game is played for the purpose of winning, an infinite game for the purpose of continuing the play." *Movement vs. institution in ten seconds.*
