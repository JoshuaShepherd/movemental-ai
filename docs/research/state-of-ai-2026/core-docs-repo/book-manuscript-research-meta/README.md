# AI Archive

This directory is a living archive of Josh's AI-related thinking, experiments, and practical work extracted from the ChatGPT export corpus.

## Purpose
- Preserve high-signal ideas about AI, agents, ChatGPT, and prompt engineering.
- Track how these ideas change over time.
- Convert raw conversations into book-ready material with evidence.

## Navigation
- `00-index/master-map.md`: top-level map of themes.
- `00-index/timeline-ai-thinking.md`: chronological evolution (newest first).
- `01-themes/`: synthesized notes by topic.
- `02-book-material/`: thesis, chapters, stories, and quote bank.
- `03-evidence/`: source log and claim-to-evidence mapping.
- `04-open-questions/`: unresolved tensions and research gaps.
- `05-run-reports/`: what was processed in each run.

## Update Standard
- Start with the most recent conversation exports.
- Prioritize concrete statements from user prompts.
- Add source references for every non-trivial claim.
- Extend existing notes instead of overwriting valuable synthesis.
