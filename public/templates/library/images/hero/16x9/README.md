# Hero 16:9 (full-bleed)

Template-ready hero images at **16:9** for full-bleed layouts with left-side text overlay.

## Specs

| Property | Value |
|----------|--------|
| Aspect ratio | 16:9 |
| Target size | 1920×1080 px (or 1280×720 for smaller) |
| Format | WebP |
| Focal area | **Subject/content on the right two-thirds**; **calm negative space on the left third** for headline overlay |

## Used by templates

- **Grassroots movement** — full-bleed hero, content overlay left
- **Institutional** — hero band/strip
- **Editorial light** — hero band
- **Behance-style** — full-bleed hero
- **Dark first** — art bg + portrait right
- **Dave Ferguson editorial** — full-bleed hero

## Naming

Place cropped files here with names that indicate author or source, e.g.:

- `alan-grassroots-16x9.webp`
- `brad-institutional-16x9.webp`
- `dave-editorial-16x9.webp`

Generated crops from the script use: `{author}-hero-16x9.webp` or `{source-basename}-16x9.webp`.
