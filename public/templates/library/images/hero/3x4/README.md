# Hero 3:4 (portrait panel)

Template-ready hero images at **3:4** for layouts with portrait panel on the right (or left) and text beside.

## Specs

| Property | Value |
|----------|--------|
| Aspect ratio | 3:4 |
| Target size | 600×800 px (or 900×1200 for @2x) |
| Format | WebP |
| Focal area | **Subject weighted to one side**; **room for text layout beside** the image |

## Used by templates

- **Scholarly editorial** — portrait panel right, text left (`home-hero__inner--portrait-right`)
- **Pastoral warm** — portrait or scene beside copy
- **Minimal contemplative** — portrait circle + optional band

## Naming

- `alan-hero-3x4.webp`
- `brad-hero-3x4.webp`
- `dave-hero-3x4.webp`

Or by source basename: `{source-basename}-3x4.webp`.
