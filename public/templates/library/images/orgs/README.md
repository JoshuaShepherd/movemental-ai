## Logo Specifications

### Technical Requirements
- **Format:** PNG with transparent background (preferred) or SVG
- **Minimum Width:** 500px (for retina displays)
- **Aspect Ratio:** Maintain original aspect ratio
- **Color Space:** RGB for web
- **Optimization:** Compress for web without losing quality

### Naming Convention
- Lowercase filenames
- Use hyphens (not spaces or underscores)
- Include organization name and 'logo'
- Format: `[organization-name]-logo.[extension]`

### File Size Guidelines
- Aim for under 100KB per logo
- Use tools like TinyPNG or ImageOptim for compression
- Balance quality with file size

## How to Obtain Logos

### Method 1: Official Brand Assets
Many organizations provide official logos in their press kits or brand guidelines:
- Check the organization's website footer for "Press" or "Brand Assets"
- Look for a /press or /media section
- Search for "brand guidelines" or "media kit"

### Method 2: Direct Contact
Reach out to organizations directly for official logo files:
- Explain the purpose (featuring them on Alan Hirsch's platform)
- Request high-resolution PNG or SVG format
- Ask for any brand usage guidelines

### Method 3: Screenshot + Cleanup
As a last resort for smaller organizations:
- Take high-resolution screenshot of logo from website
- Use tools like Photoshop or Figma to clean up and isolate logo
- Ensure sufficient resolution and quality

## Brand Usage Guidelines

### General Principles
- Always respect each organization's brand guidelines
- Maintain proper spacing around logos (clearspace)
- Don't distort or alter logos
- Use appropriate background colors (check contrast)
- Don't add effects or modifications

### Display Contexts
Consider where logos will be displayed:
- Homepage organizations section (smaller display)
- Individual organization pages (larger hero display)
- Footer or sidebar links (smaller display)
- Mobile responsive sizes

### Accessibility
- Ensure sufficient contrast with backgrounds
- Provide alt text for all logo images
- Consider providing text labels alongside logos

## Legal Considerations

### Usage Rights
- Logos are typically protected by trademark
- Usage in this context (featuring partner organizations) is generally acceptable
- Ensure no modifications that could damage brand integrity
- Keep attribution or credits if required by organization
- Remove logo if organization requests

- Document where each logo was sourced
- Save any email correspondence granting permission
- Keep copies of brand guidelines
- Note any restrictions on logo usage
