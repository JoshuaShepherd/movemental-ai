# Headshots (portrait)

Template-ready headshots for avatars, course instructor, book author, podcast host.

## Specs

| Variant | Aspect | Target size | Use |
|---------|--------|-------------|-----|
| 4:5 | 4:5 | 320×400 or 400×500 | Book detail author, hero portrait |
| Square | 1:1 | 80×80, 200×200 | Avatar, podcast host, small cards |

## Used by templates

- `portrait.headshot` / `portrait.headshotAlt` in leader JSON
- Book detail author avatar: `images/alan/alan-headshot-4x5.webp`
- Podcast host avatar, course instructor image

## Naming

- `alan-headshot-4x5.webp`
- `brad-headshot-4x5.webp`
- `dave-headshot-4x5.webp`

Square crops can be named `alan-headshot-square.webp` or generated on demand.
