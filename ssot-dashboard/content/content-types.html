Content Types & Creation Rules

What we create and how it’s structured. Foundation for all generated or authored content.

Content Types (Quintessential Guides)

Each type has a blueprint in _docs/themes-content/content-chain/content-types/. Use when transforming source material (e.g. master documents) into a format.

Article: Text, reading-optimized. 500–5K words (typical 1.5–2.5K). Title, meta, excerpt, intro, sections, conclusion, SEO, related links.

Course: Structured learning. Objectives, modules, lessons, application, certification. Best for transformation and skill development.

Book: Long-form, chapters, narrative or reference. Full metadata, table of contents, consistent voice.

Video / Podcast: Script or outline, segments, hooks, calls to action. Align to same value prop and audience.

Other: Event, AI agent—see content-types README. Each guide defines purpose, best-for, key elements, length.

Creation Rules

Source material (sermons, talks, existing posts) is transformed by type-specific rules: what to include, what to optimize for, structure. Voice and audience (this SSOT) apply to all. Quality bar: theological accuracy, voice consistency, practical applicability per principles doc.
