Six-Layer Type Safety Chain

Types flow downstream only. Lock each layer before proceeding.

  1. Layer 1: Database — Drizzle schema (db/schema.ts) — Structure SSOT
  2. Layer 2: Zod schemas — lib/schemas/ — Types SSOT
  3. Layer 3: Services — lib/services/simplified/
  4. Layer 4: API routes — app/api/simplified/
  5. Layer 5: Hooks — hooks/simplified/
  6. Layer 6: UI — components/

Tech Stack (Summary)

Frontend: Next.js 16, React 19, TypeScript, Tailwind, shadcn/ui. Backend: Next.js API routes, Supabase (PostgreSQL + auth), Drizzle ORM, Zod. All tenant-scoped data includes organizationId; tenant boundaries enforced at the service layer.
