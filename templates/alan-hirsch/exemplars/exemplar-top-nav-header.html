Top navigation & header patterns

Derived from the shipped SiteNav contract (glass, no hairline under bar) and from the lightweight static shell used across site-templates/. Use A for marketing parity, B for multi-page drafts, C for dense docs or hubs with in-page anchors.

Pattern A — Glass fixed bar

Mirrors src/components/nav/site-nav.tsx: translucent card surface, strong blur, primary CTA at end, links stay muted until hover. No bottom border on the bar; the first section provides tonal edge.

Pattern C — Dual row + mega disclosure

For documentation hubs: global IA on the first row, page-level anchors on the second, optional hover/focus-within panel for grouped destinations. Tokens only; panel uses ghost lift shadow.
